g-Tooltip by darkDemon78

A tooltip plugin extension for jQuery

g-Tooltip An Extension / plugin module for jQuery for nice looking Tooltips instead of the traditional dull ones

Dependencies

jQuery latest

For Internet Explorer users its compatible with IE 9 at this stage.

Install

  1. Download the plugin in .zip or tar.gz format from the download section.
  2. Extract it inside a folder
  3. Move the g-Tooltip.js with license file and gpl.txt from the extracted folder to the directory where you keep your project's javascript files.
  4. Reference the g-Tooltip.js file in the head tag of your html markup using<SCRIPT> tag.
  5. Invoke it in your page load event method. See README for detailed explanation.

Demo

Default formatting

$('body #hoverDiv[title],body #hoverHRef[title],body #hoverSpan[title]').forgeTooltips();

var options = {
	orientation: 'bottom', backgroundColor: '#000', opacity: 0.5 
};
$('body #hoverDiv[title],body #hoverHRef[title],body #hoverSpan[title]').forgeTooltips(options);

Different Color with bottom orientation

Right oriented with no opacity

var options = {
	orientation: 'right', backgroundColor: '#A72005', opacity: 1 
};
$('#hoverDiv[title],#hoverHRef[title],#hoverSpan[title]').forgeTooltips(options);

Want to override default font color?

Sometimes we want to use lighter colors for background if the default font is light it ain't gonna look good but that's not an issue you can override the default font like this 
var options = {
	orientation: 'right', backgroundColor: '#A72005', opacity: .5, fontColor:'#000'
};
